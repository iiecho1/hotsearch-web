# 热搜聚合网站

这是一个展示各平台每日热搜条目的网站。

## 功能特点

- 支持30+个平台的热搜展示
- 可选择日期查看历史热搜
- 点击热搜条目可跳转到原始链接
- 响应式设计，支持移动端访问

## 使用方法

1. 打开 `index.html` 文件
2. 选择想要查看的平台
3. 选择日期
4. 点击"加载热搜"按钮
5. 查看热搜条目，点击可跳转到原始链接

## 数据来源

数据来自GitHub仓库：[hot_searches_for_apps](https://github.com/iiecho1/hot_searches_for_apps)
